SwimTopia Tricks

Small browser tricks for making SwimTopia reports easier to use, print, and share.

Clear times from Meet Entries By Event

Use this console snippet on the SwimTopia Meet Entries By Event report to remove the Time, Date, and Meet columns and clear relay team seed times and split times before printing.

This edits only the report that is already loaded in the browser. Reload the SwimTopia page to bring the times back.

Steps

  1. Generate the Meet Entries By Event report in SwimTopia.
  2. Open the Chrome console: Control+Shift+J on Windows, or Cmd+Option+J on Mac.
  3. Paste this code after the console prompt, then press Return.
Console snippet 
(() => {
  const timeText = /^\s*(?:(?:\d+:)?\d{1,2}\.\d{2})(?:[SYML])?\s*$/i;
  const relayBlocks = new Set();
  let clearedRelays = 0;

  document.querySelectorAll("tr.relay_team").forEach((row) => {
    relayBlocks.add(row.closest(".no-page-break"));

    const splitCell = row.querySelectorAll("td")[2];
    if (splitCell && timeText.test(splitCell.textContent)) {
      splitCell.textContent = "";
      clearedRelays += 1;
    }
  });

  relayBlocks.forEach((block) => {
    if (!block) return;

    block.querySelectorAll(":scope > span").forEach((span) => {
      if (timeText.test(span.textContent)) {
        span.textContent = "";
      }
    });
  });

  document.querySelectorAll("table.stylized tr").forEach(row => {
    [6, 5, 4, 3].forEach(i => row.children[i]?.remove());
    const spacer = row.insertCell?.(-1) || document.createElement("th");
    spacer.innerHTML = "&nbsp;";
    spacer.style.width = "60%";
    spacer.style.border = "0";
    spacer.style.background = "transparent";
    if (!spacer.parentNode) row.appendChild(spacer);
  });

  console.log(`Cleared ${clearedRelays} relay split(s) and ${relayBlocks.size} relay seed time(s), then removed Time/Date/Meet columns.`);
})();
  1. If Chrome warns you before allowing pasted console code, follow its prompt, then paste again.
  2. The relay seed times and split times should disappear, and the Time, Date, and Meet columns should be removed.
  3. Click Print and save the report as a PDF.

If anything goes wrong, reload the report page and it will return to normal.

Clear entry times from Meet Entries By Athlete

Use this console snippet on the SwimTopia Meet Entries By Athlete report to clear the Entry Time header and values and remove swimmers with no events while keeping the remaining athlete names, ages, and events in place for printing.

This changes only the report currently loaded in the browser. Reload the SwimTopia page to bring the entry times back.

Steps

  1. Generate the Meet Entries By Athlete report in SwimTopia.
  2. Open the Chrome console: Control+Shift+J on Windows, or Cmd+Option+J on Mac.
  3. Paste this code after the console prompt, then press Return.
Console snippet 
(() => {
  let removed = 0;
  let cleared = 0;

  document.querySelectorAll("table.stylized tbody tr").forEach((row) => {
    const eventCell = row.querySelector("td:nth-child(3)");
    if (!eventCell?.textContent.trim()) {
      row.remove();
      removed += 1;
    }
  });

  const clearEntryTimeHeaders = () => {
    document.querySelectorAll("th, td").forEach((cell) => {
      if (cell.textContent.replace(/\s+/g, " ").trim() === "Entry Time") {
        cell.innerHTML = "&nbsp;";
      }
    });
  };

  document.querySelectorAll("table.stylized td.time").forEach((cell) => {
    cell.querySelectorAll(".sort_key").forEach((sortKey) => sortKey.remove());

    const lines = cell.querySelectorAll(".nowrap");
    if (lines.length) {
      lines.forEach((line) => {
        if (line.textContent.trim()) cleared += 1;
        line.innerHTML = "&nbsp;";
      });
      return;
    }

    if (cell.textContent.trim()) cleared += 1;
    cell.innerHTML = "&nbsp;";
  });

  clearEntryTimeHeaders();
  setTimeout(clearEntryTimeHeaders, 100);
  setTimeout(clearEntryTimeHeaders, 1000);

  console.log(`Removed ${removed} swimmer row(s) with no events and cleared ${cleared} entry time line(s).`);
})();
  1. Swimmers with no events should disappear, and the Entry Time header and values should be blank.
  2. Click Print and save the report as a PDF.

Meet Maestro Merge Helper

Maestro Merge Helper is a Chrome extension for NVSL team data coordinators who need to combine compatible B-meet events in SwimTopia Meet Maestro.

What it helps with

Use it when a B-meet has sparse heats that can reasonably be combined across compatible events. The extension helps identify merge opportunities, move swimmers into target events, reduce total heats, and review merged events before finalizing the heat sheet.

Install

  1. Open Google Chrome. The extension is not supported in Safari, Firefox, Edge, or other browsers.
  2. Open the unlisted Chrome Web Store page for Maestro Merge Helper.
  3. Click Add to Chrome and confirm the installation if Chrome asks.
  4. After installation, open it from Chrome's extensions menu.

Recommended workflow

  1. Build the meet in SwimTopia with the normal events and entries.
  2. Open the meet in Meet Maestro using Google Chrome.
  3. Open Maestro Merge Helper from Chrome's extensions menu.
  4. Review the Proposed Merges tab. These cards identify source events that could save heats if you add a compatible letter-suffixed merge target event.
  5. Choose which proposed merges you actually want to use.
  6. Return to SwimTopia and edit the meet events to add the merge target events you want. Merge target event numbers must end with a letter, such as 6M. Any letter can be used.
  7. Make each merge target broad enough for the sources you want to combine. For example, 6M could be a mixed-age target for events 5 and 6. Cross-age-group merges are supported when the target age range contains the source age ranges.
  8. Ask the opponent to mark their entries as ready again after you change the meet events.
  9. Rebuild the meet in SwimTopia.
  10. Open the rebuilt meet in Meet Maestro and open Maestro Merge Helper again.
  11. Use the Existing Targets tab, then click Merge on each merge you want to apply. Use Hide for options you do not want.
  12. Open the Organize Heats tab and click Organize for any merged event that needs review.
  13. In the organizer, use Sort By Time, Group Boys / Girls, or Group Age Groups as needed, then check that swimmers of different ages and genders are arranged the way you want.
  14. Review the heat sheet carefully before running the meet.

Review before running

The helper can speed up meet setup, but the data coordinator still needs to confirm that swimmers ended up in the right places.

  • Review the heat sheet and event numbers.
  • Check swimmer placement, empty lanes, and unusual merged events.
  • Make sure each merge choice makes sense for both teams.

If the merged setup looks wrong, return to SwimTopia and rebuild the meet from the original meet entries.

Project source and issue tracking are available on GitHub.
